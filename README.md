# SolarShift

> **Empowering Homes and Businesses with Clean Energy Solutions**

A modern, responsive website for SolarShift, a professional solar energy installation and consulting company. This project showcases residential and commercial solar solutions, financing options, and environmental impact metrics.

## 🌟 Features

### Service Offerings
- **Residential Solar** - Custom rooftop solar systems for homeowners with up to 90% electricity bill reduction
- **Commercial Solar** - Large-scale installations for businesses and commercial properties
- **Battery Storage** - Energy storage solutions for peak hour usage and outage protection
- **Solar Maintenance** - Comprehensive maintenance plans with inspections and monitoring
- **Energy Audit** - Professional property assessment and personalized solar recommendations
- **Roof Assessment** - Comprehensive structural evaluation for solar installation feasibility

### Tools & Calculators
- **Savings Calculator** - Interactive tool to estimate annual savings based on location and electricity bill
- **Free Assessment** - No-obligation consultation with solar experts

### Financing Options
- Cash Purchase - Lowest total cost of ownership with maximum long-term savings
- Solar Lease - $0 down payment with predictable monthly payments and included maintenance
- Power Purchase Agreement (PPA) - Fixed per-kWh rates lower than utility rates

### Company Achievements
- **10,000+** Installations completed
- **$50M+** Total savings for customers
- **98%** Customer satisfaction rating
- **50** States served
- **8,500+** Tons of CO₂ offset
- **25,000+** Trees saved

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build process or dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Shohanojjaman/SolarShift.git
cd SolarShift
```

2. Open the project locally:
```bash
# Option 1: Open index.html directly in browser
open index.html

# Option 2: Use a local server (recommended)
python -m http.server 8000
# Then navigate to http://localhost:8000
```

3. The site is now running and accessible in your browser.
